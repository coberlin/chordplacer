# Chord Sheet Annotator

A web-based tool for placing chord names above syllables in PDF lyric sheets. Upload a PDF, click above a syllable, type a chord name, and export an annotated PDF.

> **Status:** Under active development. See [chord-placer-plan.md](chord-placer-plan.md) for the project plan and [chord-placer-tasks.md](chord-placer-tasks.md) for user stories and tasks.

## Tech Stack

- **Frontend:** React, Hyphen (syllable splitting)
- **Backend:** Python, FastAPI, PyMuPDF, pdfplumber
- **Testing:** Jest, React Testing Library, pytest, Playwright
- **Tooling:** uv, Ruff, ESLint, Prettier
